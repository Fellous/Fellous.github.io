Introduction

Au cours de notre 2 -ème année nous avons réalisé un projet de groupe pour l’épreuve E6 utilisant comme technologie Java. Nous avons codé en utilisant l’IDLE NetBeans que nous avions eu l’habitude d’utiliser pour nos projets de l’année précédente. Nous avons reçu un contexte d’une industrie pharmaceutique avec la proposition de développer plusieurs fonctionnalités et nous avons choisi avec mon équipe la partie " médicament ""

1) Mise en place et organisation

Pour débuter ce projet nous avons dû tout d’abord nous repartir les taches pour designer ceux qui s’occuperont de la maquette du projet et ceux qui réaliseront la base de données avec les jeux d’essai. Pour ce faire nous avons utilisé un Trello pour repartir les taches et le logiciel XD pour réaliser la maquette du projet avec les diverses fonctionnalités qui étaient demandées. Nous avons, après rectification de notre professeur, corrigé certains aspects de la maquette et de la BDD

2)réalisation

Puisque nous travaillons en mode projet à trois nous avons décidé d’utiliser pour la gestion de version GitHub pour permettre la fusion des diverses branches de notre équipe, chacun ayant pour missions certaines fonctionnalités. Contrairement aux projets graphiques que nous avons réalisé l’année dernière nous avons utilisé une base de donnée et la méthode MVC (modèle/vue/contrôleur) pour structurer le projet

3)Fonctionnalité

Après avoir réuni les fonctionnalités qui ont été faites par chaque personne de notre équipe (les médicaments, les types d’individus, les prescriptions et les contre-indications…) ,nous sommes ensuite passés par la phase de commentaire de notre code pour permettre un java doc , pour apprendre à les utiliser à l’avenir et à faciliter les modifications par des individus extérieurs
